# Deep JS Fundamentals course

## Introduction
### What is the course about?

About deep, crucial fundamental mechanisms JS. There are things that a developer must know.

### What is JS?

The pillars of JS:
1) Types:
   1) Primitives
   2) Abstract operations
   3) Coercion
   4) Equality
   5) TypeScript, Flow and etc.

2) Scope
   * Nested scope
   * Hoisting
   * Closure
   * Modules

3) Objects (Oriented)
   * **this**
   * **class {}**
   * Prototypes
   * OO vs. OLOO
